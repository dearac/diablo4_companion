# Diablo 4 Season 12 — Crafting Mechanics Reference

Researched 2026-03-20 from Maxroll, Icy-Veins, GameRant, and community sources.

## Enchanting (Occultist)

- **Purpose**: Rerolls ONE existing affix on Rare/Legendary items
- **Vendor**: Occultist (unlocked at Level 25 or first Codex aspect)
- **How**: Select one affix → presented with 2 random replacements + keep current
- **Rerolls**: Unlimited, but gold cost escalates exponentially per reroll
- **Lock-in**: Once you choose an affix slot to enchant, only that slot can be re-enchanted on that item
- **Account-bound**: Enchanted items become account-bound
- **Materials**: Gold, Veiled Crystals, Fiend Roses, Forgotten Souls
- **Greater Affix rule**: Cannot enchant INTO a Greater Affix. Rerolling a GA destroys its Greater status.
- **Tip**: Enchant BEFORE upgrading or imprinting Aspects to save gold

## Tempering (Blacksmith)

- **Purpose**: ADDS new custom affixes to Rare/Legendary non-Unique gear
- **Vendor**: Blacksmith
- **How**: Select a learned Temper Manual → random affix from manual's set (3-5 options) applied
- **Ancestral items**: Can have 2 tempered affixes
- **Rerolls**: INFINITE in Season 12 via Scrolls of Restoration (from Infernal Hordes, Dark Citadel)
- **Greater Affix**: Tempering CAN produce Greater Affixes during application or reroll
- **Materials**: Gold, Veiled Crystals, Baleful Fragments, Coiling Wards
- **Tip**: Temper BEFORE enchanting — discard if bad rolls, save enchanting investment

## Jeweler (Sockets)

- Adds gem/rune sockets to gear
- Players may need to visit Jeweler if item has fewer sockets than build requires

## Key Decision Tree for Gear Optimization

1. Does item have the right affixes? → If missing 1, consider Enchanting at Occultist
2. Does item have enough tempered affixes? → If not, Temper at Blacksmith first
3. Does item have enough sockets? → If not, visit Jeweler
4. Does item have Greater Affixes? → NEVER reroll these — they are drop-only
